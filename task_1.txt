Задание 1
==========================================================================================================================

1)   Используйте команды операционной системы Linux и создайте две новых директории – «Игрушки для школьников» и «Игрушки для дошколят»
--------------------------------------------------------------------------------------------------------------------------

karina@karina-VirtualBox:~$ mkdir "Игрушки для школьников"

karina@karina-VirtualBox:~$ mkdir "Игрушки для дошколят"

karina@karina-VirtualBox:~$ ls

 Desktop     mysite     Templates              'Игрушки для школьников'

 Documents   Pictures   test.txt

 Downloads   Public     Videos

 Music       snap      'Игрушки для дошколят'

-------------------------------------------------------------------------------------------------------------------------

2)   Создайте в директории «Игрушки для школьников» текстовые файлы - «Роботы», «Конструктор», «Настольные игры»
-------------------------------------------------------------------------------------------------------------------------

karina@karina-VirtualBox:~$ touch "Игрушки для школьников/Роботы"

karina@karina-VirtualBox:~$ touch "Игрушки для школьников/Конструктор"

karina@karina-VirtualBox:~$ touch "Игрушки для школьников/Настольные игры"

karina@karina-VirtualBox:~/Игрушки для школьников$ ls

 Конструктор  'Настольные игры'   Роботы

------------------------------------------------------------------------------------------------------------------------

3)    Создайте в директории «Игрушки для дошколят» текстовые файлы «Мягкие игрушки», «Куклы», «Машинки»
------------------------------------------------------------------------------------------------------------------------

karina@karina-VirtualBox:~$ touch "Игрушки для дошколят/Мягкие игрушки"

karina@karina-VirtualBox:~$ touch "Игрушки для дошколят/Куклы"

karina@karina-VirtualBox:~$ touch "Игрушки для дошколят/Машинки"

karina@karina-VirtualBox:~$ cd "Игрушки для дошколят"

karina@karina-VirtualBox:~/Игрушки для дошколят$ ls

 Куклы   Машинки  'Мягкие игрушки'

------------------------------------------------------------------------------------------------------------------------

4)   Объединить 2 директории в одну «Имя Игрушки»
------------------------------------------------------------------------------------------------------------------------

karina@karina-VirtualBox:~$ mv "Игрушки для школьников" "Имя Игрушки"

karina@karina-VirtualBox:~$ mv "Игрушки для дошколят" "Имя Игрушки"

karina@karina-VirtualBox:~$ ls

 Desktop     Downloads   mysite     Public   Templates   Videos

 Documents   Music       Pictures   snap     test.txt   'Имя Игрушки'

karina@karina-VirtualBox:~$ cd 'Имя Игрушки'

karina@karina-VirtualBox:~/Имя Игрушки$ ls

'Игрушки для дошколят'   Конструктор  'Настольные игры'   Роботы

------------------------------------------------------------------------------------------------------------------------

5)   Переименовать директорию «Имя Игрушки» в «Игрушки»
------------------------------------------------------------------------------------------------------------------------

karina@karina-VirtualBox:~$ mv "Имя Игрушки" "Игрушки"

karina@karina-VirtualBox:~$ ls

Desktop    Downloads  mysite    Public  Templates  Videos

Documents  Music      Pictures  snap    test.txt   Игрушки     <---------

-----------------------------------------------------------------------------------------------------------------------

6)   Просмотреть содержимое каталога «Игрушки».
-----------------------------------------------------------------------------------------------------------------------

karina@karina-VirtualBox:~$ ls Игрушки

'Игрушки для дошколят'   Конструктор  'Настольные игры'   Роботы

karina@karina-VirtualBox:~$ 

-----------------------------------------------------------------------------------------------------------------------

figma-linux 0.9.5 from YouDontHavePermission (youdonthavepermissiony) installed

копирование из каталога А в каталог Б).
